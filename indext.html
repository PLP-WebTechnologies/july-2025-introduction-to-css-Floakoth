The CSS Box Model

Every element in web design is a rectangular box. The CSS box model describes how these boxes are sized and spaced.

Box Model Components

Margin
Border
Padding
Content

Box Model Visualization

This is a box with margin, padding, and border applied.

The total width of an element is calculated as:

width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Similarly, the total height is calculated as:

height + padding-top + padding-bottom + border-top + border-bottom + margin-top + margin-bottom

Basic CSS Styling

Text Styling

This paragraph demonstrates basic text styling with color, font-size, and line-height properties.

Font Properties Example:

p {
  font-family: 'Arial', sans-serif;
  font-size: 16px;
  line-height: 1.5;
  color: #333333;
}

Backgrounds & Borders

CSS allows you to style backgrounds and borders of elements.

Background Example:

.element {
  background-color: #f0f0f0;
  border: 2px solid #3498db;
  border-radius: 5px;
}

Box Shadow Example:

.card {
  box-shadow: 0 4px 8px rgba(0, 0, 0, 0.1);
}

Practical Examples

Buttons with Box Model

Buttons are a great example of the box model in action:

button {
  padding: 12px 25px; /* Inside spacing */
  margin: 0 10px; /* Outside spacing */
  border: none; /* No border */
  border-radius: 5px; /* Rounded corners */
}

Spacing Elements

Using margin and padding to create space between elements:

This element has padding inside and margin below.

Another element with similar spacing.

.element {
  margin-bottom: 15px; /* Space below */
  padding: 15px; /* Space inside */
}
